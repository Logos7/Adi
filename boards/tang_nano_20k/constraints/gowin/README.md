# Gowin constraints for Tang Nano 20K

This directory contains full Gowin `.cst` files and reusable constraint fragments.

Fragments are kept under:

```text
boards/tang_nano_20k/constraints/gowin/fragments/
```

Full ready-to-use files may be generated or maintained beside them:

```text
tang_nano_20k_uart.cst
tang_nano_20k_uart_hdmi.cst
```
